# 📱 TinDevs APK - Instrucciones de Instalación

## 📦 Archivos Disponibles

### 🔵 TinDevs_App_Principal_v1.0.0.apk (49MB)
**Para:** Postulantes (Desarrolladores) y Empleadores
**Funciones:**
- Registro como postulante o empleador
- Swipe de propuestas laborales
- Chat entre matches
- Sistema de calificaciones
- Gestión de perfil

### 🔴 TinDevs_Admin_v1.0.0.apk (49MB)
**Para:** Administradores del sistema
**Funciones:**
- Dashboard administrativo
- Validar propuestas de empleadores
- Gestionar usuarios
- Visualizar documentos de validación
- Eliminar contenido inapropiado

## 📲 Instalación

### Requisitos:
- Android 5.0 (API 21) o superior
- Aproximadamente 100MB de espacio libre
- Conexión a internet

### Pasos:
1. **Habilitar instalación de fuentes desconocidas:**
   - Configuración → Seguridad → Fuentes desconocidas (ON)
   - O Configuración → Aplicaciones → Acceso especial → Instalar apps desconocidas

2. **Descargar e instalar:**
   - Transferir el APK al dispositivo Android
   - Abrir el archivo .apk desde el explorador de archivos
   - Seguir las instrucciones en pantalla

3. **Primer uso:**
   - Crear cuenta nueva o iniciar sesión
   - Completar perfil según el tipo de usuario
   - ¡Comenzar a usar TinDevs!

## ⚠️ Notas Importantes

- Estos son builds de desarrollo/testing
- Requiere conexión a Firebase para funcionar
- Los datos se almacenan en la nube
- Si encuentras bugs, reportarlos al desarrollador
